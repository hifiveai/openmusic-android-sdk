HIFIVE音乐播放器（含海量正版音乐的API）
=======================

## 介绍

&emsp;&emsp;HIFIVE音乐播放器是一款Android、iOS两端的客户端系统，由成都嗨翻屋科技有限公司打造的播放器demo，开源无加密，支持二次开发。
该demo包含了700万+海量正版音乐的API，包含了电台、歌单、搜索歌曲、热歌推荐、歌曲播放、歌词同步等功能。获取音乐API的权限可通过注册HIFIVE音乐开放平台申请授权。
